# 프로젝트 계획서

## 1. 프로젝트 주제

내가 만들 프로그램의 주제: BMI와 에상 체지방률 계산 및 운동, 식단 추천기

예) 기분과 시간에 따른 활동 추천기

---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.

예) 사용자가 현재 기분과 사용할 수 있는 시간을 입력하면, 조건에 맞는 활동을 추천해 주고 싶다.

내 설명:사용자의 인적사항을 입력하면 BMI지수와 예상 체지방률을 알려주고 그것에 따른 운동 및 식단을 추천해준다

- 
- 

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 사용자가 입력한 신장(cm)을 미터(m) 단위로 변환한 후, 체중 / (신장 * 신장) 공식을 적용해 BMI 지수를 계산함.

학술적으로 검증된 Deurenberg 공식인 (1.20 * BMI) + (0.23 * 나이) - (10.8 * 성별 상수) - 5.4를 적용하여 예상 체지방률을 연산함.

성별 기준치(남성 20% 이상, 여성 28% 이상)에 따라 과체중 위험 여부를 판정하고, 이에 해당할 경우 성별 맞춤형 운동 루틴 및 건강 가이드를 매칭함.

## 4. 2차원 리스트 설계

내 프로그램에서 사용할 2차원 리스트의 이름: single_user_matrix

예) foods, movies, characters, questions, records

```python

single_user_matrix = [
    ["홍길동", "남성", 25, 175.0, 70.0]
]

### 한 행의 의미
한 명의 사용자가 입력한 고유의 신체 정보 및 개인 인적 사항 데이터 리스트([이름, 성별, 나이, 신장, 체중])를 의미함.

### 각 열의 의미

| 열 번호 | 의미 | 예시 |
|---:|---|---|
| 0번 열 |사용자의 이름 (문자열)  | "홍길동" |
| 1번 열 | 사용자의 성별 (문자열) |  "남성" 또는 "여성"|
| 2번 열 | 사용자의 나이 (정수형) | 25 |
| 3번 열 | 사용자의 신장 (실수형, cm 단위) | 175.0 |
| 4번 열 |사용자의 체중(실수형,kg단위) | 70.0|

## 5. 함수 설계

| 함수 이름 | 역할 | 입력값 | 반환값 또는 출력 |
|input_single_user()|사용자 정보를 입력받아 데이터 검증 후 2차원 리스트 생성|-없음 (콘솔 직접 입력)|2차원 리스트 (single_user_matrix)|
| calculate_metrics() | Deurenberg 공식을 기반으로 체지방률 계산 |  bmi (실수), age (정수), gender (문자열)| 계산된 체지방률 (실수, 반올림) |
| recommend_exercise() |  과체중 위험군 대상 성별 맞춤 운동 가이드 출력| gender (문자열) | 콘솔 창에 운동 가이드 문구 출력 |
| process_and_print_result()|  2차원 리스트를 순회하며 전체 연산 수행 및 분석 리포트 출력| user_data (2차원 리스트) |  개인 신체 정보, BMI, 체지방률, 판정 및 운동 가이드 출력|

## 6. 사용할 제어구조

### 조건문을 사용할 부분

예) 사용자의 예산보다 가격이 낮은 항목만 추천한다.

내 계획:calculate_metrics() 및 recommend_exercise() 함수 내에서 사용자의 성별이 '남성'인지 '여성'인지 판단하여 연산 상수와 운동 추천 문구를 다르게 적용할 때 사용함.

process_and_print_result() 함수 내에서 계산된 체지방률이 성별 비만 기준치(남성 20%, 여성 28%)를 넘었는지 확인하여 '과체중 위험' 판정을 내리고 운동 추천 함수를 호출할 때 사용함.

프로그램 종료 직전 사용자가 '예'를 입력했는지 '아니요'를 입력했는지 판별하여 반복을 계속할지 종료할지 결정할 때 사용함.


### 반복문을 사용할 부분

예) 2차원 리스트의 모든 항목을 하나씩 검사한다.

내 계획:프로그램 실행 메인 루틴에서 while True 무한 루프를 사용하여, 한 사람의 계산이 끝난 후 사용자가 '예'를 누르면 프로그램이 꺼지지 않고 처음으로 돌아가 새로운 사람의 정보를 입력받을 수 있도록 제어함.

process_and_print_result() 함수 내부에서 for row in user_data: 반복문을 사용하여 2차원 리스트 내부에 저장된 행(사용자 데이터)을 추출하고 내부 원소들을 제어할 때 사용함.

## 7. 예상되는 예외 상황

예) 조건에 맞는 추천 결과가 없을 때, 잘못된 입력을 했을 때 등

1. 사용자가 신장, 체중, 나이를 입력할 때 숫자 외에 단위 문자(cm, kg, 세 등)를 함께 입력하는 경우

대응 계획: 코드 내에 try-except ValueError 예외 처리 구문을 두어, 숫자가 아닌 문자가 들어왔을 때 프로그램이 비정상 종료(팅김 현상)되지 않도록 막고 올바른 입력 안내 메시지를 출력함.

2. 성별 입력창에 '남성/여성'이 아닌 잘못된 단어나 공백, 오타(남자, 여 등)를 입력하는 경우

대응 계획: if gender not in ["남성", "여성"] 조건문 검증 단계를 두어 잘못된 텍스트가 유입되었을 시 에러 메시지를 띄우고 즉시 함수를 안전하게 리턴(return)시킴.
