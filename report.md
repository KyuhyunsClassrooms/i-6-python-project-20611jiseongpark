# 최종 보고서

## 1. 프로그램 주제와 목적

내 프로그램의 주제: BMI와 에상 체지방률 계산 및 운동, 식단 추천기

이 프로그램을 만든 이유: 헬스장이나 병원을 가지 않으면 BMI 지수나 체지방률을 알지 못하고 그에 따른 운동 방법이나 루틴을 추천받아 효율적으로 건강한 사람들이 몸을 가질 수 있도록 만들고 싶기 때문이다


---

## 2. 문제 분해 및 모델링

### 입력
이름, 나이, 성별,신장, 몸무게

### 처리
체지방량 계산 공식 등을 사용하여 BMI와 예상 체지방률을 계산하고 체지방이 정상인지 아닌 지 계산하고, 운동을 추천해줌

### 출력
예상 체지방률과 BMI지수 및 체지방률이 정상이 아니면 운동을 추천해준다

---

## 3. 사용한 2차원 리스트 설명

2차원 리스트 이름:single_user_matrix

한 행의 의미:한 명의 사용자가 입력한 고유의 신체 정보 및 개인 인적 사항 데이터 리스트([이름, 성별, 나이, 신장, 체중])를 의미함.

각 열의 의미: 0번 열사용자의 이름 (문자열)"홍길동"1번 열사용자의 성별 (문자열)"남성" 또는 "여성"2번 열사용자의 나이 (정수형)253번 열사용자의 신장 (실수형, cm 단위)175.04번 열사용자의 체중 (실수형, kg 단위)70.0


이 2차원 리스트를 사용한 이유:
1차원 리스트를 사용할 경우 이름, 성별, 나이, 키, 몸무게 등의 데이터가 개별적으로 분산되어 데이터 오염이나 유실의 위험이 있습니다. 반면, 대괄호를 중첩한 2차원 리스트 구조([[이름, 성별, 나이, 키, 몸무게]])를 채택함으로써, 엑셀 표의 한 행(Row)처럼 한 사람의 신체 정보를 데이터의 유기적 연관성을 가진 하나의 묶음으로 안전하게 관리할 수 있습니다. 이는 추후 for 반복문을 통해 각 열(Column)의 데이터를 순서대로 추출하고 연산하는 과정에서 프로그램의 구조적 안정성과 확장성을 극대화하는 역할을 합니다.

## 4. 함수 설명
함수 이름역할입력값반환값 또는 출력input_single_user()사용자 정보를 입력받아 데이터 검증 후 2차원 리스트 생성없음 (콘솔 직접 입력)2차원 리스트 (single_user_matrix)calculate_metrics()Deurenberg 공식을 기반으로 체지방률 계산bmi (실수), age (정수), gender (문자열)계산된 체지방률 (실수, 반올림)recommend_exercise()과체중 위험군 대상 성별 맞춤 운동 가이드 출력gender (문자열)콘솔 창에 운동 가이드 문구 출력process_and_print_result()2차원 리스트를 순회하며 전체 연산 수행 및 분석 리포트 출력user_data (2차원 리스트)개인 신체 정보, BMI, 체지방률, 판정 및 운동 가이드 출력
가장 중요한 함수 하나를 골라 자세히 설명하세요.

함수 이름: calculate_metrics()

설명:이 함수는 프로그램의 실질적인 '연산 알고리즘'을 담당하는 뇌의 역할을 수행합니다. 입력받은 사용자의 신장과 체중을 활용하여 BMI를 도출할 뿐만 아니라, 성별 조건문(if-else)을 가동하여 남성과 여성에게 각각 다르게 적용되는 학술적 상수값(Deurenberg 공식)을 계산 인터페이스에 매칭합니다. 아무리 훌륭한 입력 시스템과 출력 포맷이 갖추어져 있더라도, 실제 체지방률이라는 데이터 가치를 창출하는 이 핵심 연산 함수가 없다면 프로그램의 목적성이 상실되므로 본 프로젝트에서 가장 중요한 함수로 정의할 수 있습니다.

## 5. 조건문과 반복문 활용

### 조건문 사용 부분

사용자가 성별을 입력할 때 '남성' 또는 '여성'으로 올바르게 입력했는지 검사함.

체지방률을 계산할 때 사용자의 성별이 '남성'인지 '여성'인지 검사하여 공식에 적용할 상수를 다르게 지정함.

계산된 체지방률이 성별 기준치(남성 20% 이상, 여성 28% 이상)를 초과하는지 검사하여 과체중 여부를 판정함.

프로그램이 종료되기 전 사용자가 '예' 또는 '네'를 입력했는지 검사하여 재실행 여부를 판단함.


### 반복문 사용 부분

메인 루틴 내에서 while True 무한 루프를 사용하여 한 사람의 체지방률 분석이 끝난 후 사용자가 원할 때까지 전체 입력 및 연산 과정을 무한 반복 처리함.

process_and_print_result() 함수 내에서 for row in user_data:문을 사용하여 2차원 리스트 행렬에 담긴 고유 신체 정보 데이터를 행 단위로 추출하는 과정을 반복 처리함.


### 조건문과 반복문이 함께 작동한 부분

프로그램 지속 여부 제어 파트: while True 반복문 내부에서 한 사이클의 분석이 완료된 후, 사용자의 답변을 입력받아 if retry in ['예', '네'] 조건문으로 판별함. 사용자가 '예'를 입력하면 반복문의 처음으로 돌아가고(continue), '아니요' 등 다른 값을 입력하면 조건문에 의해 반복문을 탈출(break)하도록 설계되어 두 제어구조가 상호작용함.

체지방 판정 및 운동 추천 파트: for 반복문을 통해 2차원 리스트에서 신체 정보를 꺼내 읽는 도중, 내부에서 if 조건문을 가동하여 해당 사용자가 과체중 위험군에 속하는지 판정하고 맞춤형 운동 추천 함수를 호출하도록 구현함.

## 8. Gemini 활용 내용 요약

Gemini에게 도움받은 부분: 코드와 프로그램을 작성하는데 도움을 받음


Gemini가 제안했지만 내가 수정하거나 사용하지 않은 부분: 제미나이가 딱히 제안한 것이 없음


최종 코드에서 내가 직접 판단하거나 수정한 부분: 아이디어를 냈고 그 아이디어로 제미나이가 코드를 짜줌 ㄸ딱히 판단하거나 수정한 부분이 업ㄳ음
